## Instrucciones de Uso
1. Selecciona la skill adecuada según la tarea.
2. Copia el contenido completo y úsalo como system prompt o contexto para el agente.
3. El agente ahora "sabrá" cómo interpretar los datos y dónde encontrar cada componente del sistema.
