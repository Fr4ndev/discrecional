# 🧠 Manual de Estrategia: Dual-Regime CCXTV2

Este manual define la operativa cuando el sistema detecta señales contradictorias entre el **Timeframe Diario (Local)** y el **Timeframe Mensual (Macro)**.

## 1. La Paradoja de los Z-Scores
- **Daily 100/100 + Monthly 0/100:** Es el escenario de "Rebote de Gato Muerto" o capitulación temporal.
- **Acción:** Operar el Long con 50% de sizing y cerrar en niveles de Fibonacci (0.5 o 0.618). No buscar "To the Moon".

## 2. Niveles de Invalación Cruzada
- Si el precio rompe el **Monthly Stop Loss ($81,250)**, el régimen macro ha cambiado de Distribución a Re-Acumulación. **Cerrar todos los Shorts.**
- Si el precio rompe el **Daily Stop Loss ($75,450)**, el rebote ha fallado y el Short Macro se acelera. **Full Send en Short.**

## 3. Indicadores de Confirmación
- **Toxicity > 0.70:** Confirma la dirección del breakout.
- **Basis < -0.05%:** Confirma que el Spot está liderando (Bullish).
- **Basis > +0.05%:** Confirma que el Perp está liderando (FOMO/Bearish).
